<a href="https://hackheatharu.xyz/update-consideration-of-splatoon2-ver-4-2-0/">前回</a>から全然、日が経っていないので半分不意打ちでした。

&nbsp;

内容は主に
<ul>
 	<li>新武器４種類追加</li>
 	<li>べっちゅーコレクション第４弾の発表</li>

&nbsp;
<h2><strong>『メイン性能アップ』の詳細</strong></h2>
{{< tweet 1066280921211449344 >}}
&nbsp;

<strong>『メイン性能アップ』</strong>はアップデート後、ショップなどでこのパワーが付いたギアを購入できるとのこと。

とりあえずサーモンランやりまくってギアを集めたいところですね。

もしくはスーパーサザエが有り余っている人もいるのではないでしょうか。

かくいう私も200以上余っているので、このギアパワーが有用であれば集めることに注力すると思います。

&nbsp;

さて、具体的にはどのような効果があるんでしょうか。

以下の画像から情報を抜粋してみます。

&nbsp;

&nbsp;

<img class="alignnone size-medium wp-image-1176" src="https://hackheatharu.xyz/wp-content/uploads/2018/11/Dswwue_VsAE8WZX-300x169.jpg" alt="" width="300" height="169" /><img class="alignnone size-medium wp-image-1177" src="https://hackheatharu.xyz/wp-content/uploads/2018/11/DswwvWvV4AAJWUb-300x169.jpg" alt="" width="300" height="169" /> <img class="alignnone size-medium wp-image-1179" src="https://hackheatharu.xyz/wp-content/uploads/2018/11/DswwwH2UcAAyoqv-300x169.jpg" alt="" width="300" height="169" />

&nbsp;
<ul>
 	<li>ジャンプして射撃した時の弾がブレにくくなります。[スプラシューター]</li>
 	<li>塗り性能がアップします。[プロモデラーMG]</li>
 	<li>与えるダメージが大きくなります。[スプラローラー]</li>
 	<li>射撃継続時間が長くなります。[バレルスピナー]</li>
 	<li>傘が復活するまでの時間が短くなります。[スパイガジェット]</li>
 	<li>塗りながら進む速度がわずかにアップします。[パブロ]</li>
</ul>
&nbsp;

といった感じで、ギア名の通り『<strong>メインの個性を伸ばす</strong>』といたところでしょうか。

しかも、武器毎に上がる性能が設定されていることに驚きました。

普通であれば『シューター系なら～』『ローラー系なら～』という具合だと思うのですが、ちゃんと<strong>個別に違う能力が上がる</strong>みたいですね。

上記で言うならスプラシューターとプロモデラーMGは同じブキカテゴリーですが、前者はジャンプ撃ちの精度、後者は塗り性能が上がります。

&nbsp;

おそらくこのギアによってプロモデラーは、<strong>かつて</strong><a href="https://hackheatharu.xyz/bad-locust/"><strong>強ブキイナゴに奪われた栄光</strong></a>を取り戻せるようになるのかもしれませんね。

<strong>ただ、何個かは上がる性能が同じになると予想しています。</strong>

プライムとジェットスイーパーなんて同じような感じになるんじゃないかなと思うんです。

スプラローラーとカーボン、ダイナモとヴァリアブルみたいな、ある程度似た者同士は同じ能力が上がると私は予想します。

&nbsp;

&nbsp;
<h2>スパッタリー視点で考える</h2>
さて、この『メイン性能アップ』に関してスパッタリーの視点で考えていきたいと思います。

&nbsp;
<h4>シューターが相対的に強くなる可能性</h4>
まずスプラシューターのように、<strong>シューターのジャンプ撃ち精度が上がることは脅威</strong>ですね。

マニュ系はシューター対面に有利をとれます。

個人的には、シューターは接近されたマニュに対してジャンプ撃ちで対処しようとするためカモなんですよね。

このカモだった要因の１つ『ジャンプ撃ちの精度の悪さ』が解消されてしまいます。

&nbsp;

スプラシューターはオーソドックスに戦えるブキですからね。

AIMのうまい人がつけたら確実に仕留められることでしょう。

私はやっかいだなと思ってみています。

ギア何個でどれくらい修正されるかはわかりませんが、上位勢は最低限の運用分を必ずつけるだろうと予想しています。

&nbsp;
<h4>スパッタリーはメインの何が向上するのか</h4>
さて今回の記事の本題といってもいい部分です。

我が愛ブキ『スパッタリー』には、どのような恩恵が施されるのでしょうか。

ツイッターで考えた限りでは、ざっとこんな感じでした。

&nbsp;

{{< tweet 1066284798690250753 >}}
ツイッターではマニュと記載していますが、この時はブキカテゴリーでの性能アップだと勘違いしていました。

しかし、ブキ毎に性能が設定されていることが改めてわかったので、ここではスパッタリーに限った考察をしていきます。

&nbsp;

叶うなら、私は以下のいずれかを所望します。
<ol>
 	<li><strong>スライド速度UP</strong></li>
 	<li><strong>スライド硬直軽減</strong></li>
 	<li><strong>スライド後の射撃威力UP</strong></li>
</ol>
&nbsp;

１は『まるで瞬間移動したかのようなスライド』がしてみたいだけですね。

<strong>今でも十分早いのですが、もっと速くできるなら使ってみたいです。</strong>

&nbsp;

他のフォロワーさんとかのTLを見てみると、２の意見が多くみられた気がしました。

ただこれ、実現したらクアッドの十八番を奪ってしまうような気がするんですよね。

<strong>疑似的に4回、下手したらそれ以上のスライドが可能になってしまいますから。</strong>

&nbsp;

<strong>個人的に３が来たら熱いと思っています。</strong>

スプラローラーにも同じ効果があるので、可能性がないわけでもないと思います。

3.9積んだら96ガロンと同じ威力『62.0』になってほしいです。

ホコ割り最速の座を奪還できるのはもちろんですが、<strong>威力減衰のダメージが31</strong>になるのがデカいですね。

最近スパッタリー使っててムカつくのが『威力減衰のせいで倒しきれない』ことが多すぎなことです。

というか射程が環境を支配してて近づくのさえ困難なんですから、せめて減衰くらい何とかしてほしいという願いがあったんです。

なので正直３来てくれないかなぁと凄く願っています。

&nbsp;

&nbsp;
<h4>最後のアプデとして</h4>
今回の4.3.0が最後のアプデになるそうです。

巷では『黒スパッタリー』を願う声も結構ありますね。

確かに出たら私も凄く嬉しいのですが、最後の望みである『<strong>マニューバーの当たり判定を戻す修正</strong>』をしてほしいです。

<strong>最悪、メイン性能アップを3.9積んだら今と同じような当たり判定になるのは許してあげますから、本当に頼むから戻してください</strong>。

では、12月5日を期待しましょう。
