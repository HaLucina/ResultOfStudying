スマブラSPからファイアーエムブレム（以下FE）を知った方に向けて『<strong>FEキャラの紹介</strong>』をする記事です。

今回は<strong>アイク</strong>について紹介していきます。

FEの世界では、このキャラはどのような活躍をしていたのかという解説を私なりにしていきます。

また、<strong>作品のネタバレも躊躇なく記載</strong>していきますので注意してください。

&nbsp;

&nbsp;
<h2>スマブラのアイク</h2>
![test](https://res.cloudinary.com/ddghc4l09/2018/12/maxresdefault-31.pagespeed.ce_.7xDYIGLuUF.jpg)
スマブラXで初参戦しました。

アイクは全FEファイターで『<strong>最も攻撃力が高い</strong>』です。

１つ１つの攻撃範囲も広いという反面、発生の速い攻撃が少ないという短所があります。

スーパーアーマーのある『<strong>天空</strong>』や比較的早い弱攻撃を織り交ぜながら、強力なスマッシュ攻撃につなげる必要があります。

&nbsp;

【<strong>アイクの長所</strong>】
<ul>
 	<li>攻撃力が高い</li>
 	<li>攻撃範囲が広い</li>
</ul>
&nbsp;

【<strong>アイク</strong><strong>の短所</strong>】
<ul>
 	<li>技が出るまで遅く隙が大きい</li>
 	<li>移動速度は遅め</li>
</ul>

<h2>FEでのアイク</h2>
![test](https://res.cloudinary.com/ddghc4l09/2018/12/canva-photo-editor-4-1.png"><img class="alignnone size-medium wp-image-1665" src="https://hackheatharu.xyz/wp-content/uploads/2018/12/canva-photo-editor-4-1-300x282.png" alt="" width="300" height="282" /></a><a href="https://hackheatharu.xyz/wp-content/uploads/2018/12/canva-photo-editor-9.png)
(画像引用：<a href="https://www.nintendo.co.jp/fe/characters/detail/marth04.html">ファイアーエムブレムワールド【FIRE EMBLEM WORLD】</a>)

&nbsp;

2005年4月20日、ゲームキューブから発売された『<strong>ファイアーエムブレム 蒼炎の軌跡</strong>』の主人公がアイクです。

不愛想で口下手なところがありますが、アイクが所属するグレイ傭兵団の団員とは家族同然の強い絆を持っています。

種族や貴族関係なく、差別しないで誰に対しても対等に振舞います。

一見クールなようで、実は父親であるグレイル譲りの熱血漢な描写もあります。

&nbsp;

ベオクとラグズ（平たく言えば人と獣）という種族で分け隔てられた国家で成り立つテリウス大陸で、父親が団長を務めるグレイル傭兵団の一員として身を置いていました。

デイン王国からの侵攻によって亡命中だったエリンシア姫を偶然にもグレイル傭兵団が保護します。

亡命先であるラグズが治める国家ガリア王国へ向かう途中、父グレイルが漆黒の騎士との一騎打ちで命を落とします。

その後、アイクは傭兵団を継ぎ、妹のミストが持つ亡き母の形見”炎のメダリオン”をめぐるデインとの波乱な運命へと導かれていきます。

&nbsp;

2007年2月22日にはWiiから続編の『<strong>ファイアーエムブレム 暁の女神</strong>』が発売されました。

物語は4部構成となっており、御馴染のマッチョでゴリラなアイクは第3部から主人公を務めます。

永い眠りから目覚めた【正の女神】アスタルテが醜く争う人々に裁きを下し、ほぼ全ての人類が石と化した世界を救うため戦います。

因縁の相手、漆黒の騎士とは本作で決着をつけることができます。

女神との戦いを終えた後、別天地を目指して旅へ出ます。

やがてアイクは『<b>蒼炎の勇者</b>』としてテリウス大陸で末永く語り継がれていく伝説となるのです。

&nbsp;

&nbsp;
<h2>FEのキャラ性能</h2>
蒼炎のアイクはFEシリーズでも最強の主人公という呼ばれることもあります。

ステータスの高さはもちろんですが、アイクの代名詞ともいえる奥義『<strong>天空</strong>』は、発動したら相手をほぼ一撃で葬れる必殺です。

そして彼の専用武器『<strong>ラグネル</strong>』の性能もあり<strong>最高難易度でも敵をなぎ倒せる無双っぷり</strong>をゲームで味わうことができます。

&nbsp;

しかし、そのためか暁のアイクは少し弱体化してしまいました。

特に魔防面が著しい調整を施されており、いくら関節攻撃できるからといって前に出すぎると魔法でやられてしまう欠点が生まれました。

特に痛いのは<strong>『天空』の発動率が蒼炎の半分</strong>に調整されたことで、前作のように無双することは厳しいです。

暁では剣の他に斧も使えるようにもなります。

&nbsp;

&nbsp;
<h2>アイク雑学</h2>
<ul>
 	<li>前々作『<strong>ファイアーエムブレム 烈火の剣</strong>』の主人公ヘクトルが好評だったことと、製作スタッフからの要望も相まって『より男らしさを追求した』キャラクターとして生まれた。</li>
 	<li>暁アイクのファンの通称は、見た目がゴリラになったことと獣のラグズを足した『<strong>ゴリラグズ</strong>』で親しまれている。
ドンキーコングと一緒に登場する時にそう呼ばれる理由がコレである。</li>
 	<li>『<strong>誰も死ぬな！</strong>』などの熱い名言がある中、アイクで一番人気のあるセリフは『<strong>ぬぅん！</strong>』</li>
</ul>
